How to Make a Scratch Project

By Toydotgam


If you have always wanted to code but don't know how, Scratch is a good place to start!




Step 1. Creating a Project

Welcome to Scratch! If you haven't used it before, don't worry! Scratch is an amazing place for programmers taht have just started coding, like kids.


First things first, open the main editor in Scratch. Now, try and import some sprites, these are the main functional parts of the program. Once you have some imported from the Scratch library, look into the "Events" tab, place a "When Green Flag Clicked" block at the top of the editor page. This block will activate when the code is set off.


Step 2: Making it Work

Go to the "Motion" tab and move a "move (10) steps" block onto the page and connect it to the bottom of the Green Flag block, a bit like LEGOR! Once the Green flag at the top right of the stage is clicked, your sprite should move however many steps you set fir it to move.


Step 3: Experiment!

Scratch is endless! Expleriment to you heart's desire, you can change how the sprile look in the "Looks" tab. Do whatever you want, join the Scratch community.

Some people have even made spin-off of popular games, like griffpatch, he made his own "'Paper' Minecraft"! Amazing!


Step 4: The Next Step

After explerimenting with Scratch the easy way, go to Python, find out about it yourself! Many coding languages of plain text await not just Scratch's brick-layout. Try HTML (Hyper Text Markup Language), that is the code they used to make this website.

Have a go at some HTML:

You'll have to start with the HTML Header, remember to put it in angled brackets: <, >:

!DOCTYPE html


Then you'll need to add the 'head' element and the 'body' one as well.

Then add '/head' and '/body' as well and put a 'p' in there, (paragraph) add a '/p' also. Have a loock at some basic websites like example.com. Press F12 and have a look at the HTML code, there are plenty of books and documentation to help you on your journey.

Good Luck!

Scratch is an internet software, click this link to go to the Scratch website, or download Scratch 3.0 from here.

Scratch 2.0 may not run on older machines running an OS below Windows Vista or Mac OS X 10.5. Scratch 1.4 can be downloaded here for Windows 2000, XP, Vista, 7 & 8 and can be downloaded for Mac OS X 10.4 or later.

If your project doesn't do what you want it to do, read through the blocks an see where the lines went wrong and how. Then remove or fix the error by changing some of the blocks around.
